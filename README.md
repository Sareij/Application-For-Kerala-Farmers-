# **VIDHAI – Smart Agriculture Assistant**

**VIDHAI** is an AI-based agriculture platform that assists farmers and students with **crop recommendation and plant disease detection** using machine learning and a web-based interface.

## Features

* **Crop Recommendation System** – Suggests suitable crops based on soil and environmental data.
* **Plant Disease Detection** – Identifies diseases from leaf images using a trained ML model.
* **Web Application** – Simple and user-friendly interface for result visualization.

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **Machine Learning:** TensorFlow / Keras
* **Version Control:** Git & GitHub
